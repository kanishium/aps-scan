--Security Scan Dashboard (Frontend)

A responsive frontend implementation of a Security Scan Dashboard built using React and Tailwind CSS.
The application simulates a cybersecurity scanning platform with authentication, a dashboard overview, and an active scan detail interface.

Tech Stack

1-React

2-Tailwind CSS

3-React Router DOM

4-React Icons

5-React Hot Toast

6-Vite

--Features

1-Responsive signup page with form validation

2-Dashboard with scan statistics and scan list

3-Search and filter functionality for scans

4-Active scan detail page with console logs and findings

5-Global dark/light theme toggle

6-Fully responsive layout (mobile and desktop)

7-Interactive UI elements with toast notifications

--Known Limitations

1-No backend integration (mock JSON data used)

2-Authentication and scan actions are simulated

3-Social login buttons are UI-only implementations
